SNarVIS AI 에이전트(훈수 AI)

AI 질문/풀이 + 데이터 분석을 통합한 지능형 학습 어시스턴트

실시간 학습 진단 개인화 추천 엔진 학습 패턴 분석

① AI 질문 · 풀이 SNarGPT / SNargo

수능 특화 엔진으로 질문을 받아 이해·풀이·설명까지. 인식(초지각) 개선과 과목 확장, 유사문제 출제가 핵심 축입니다.

SNarGPT

역할: 질문 접수 · 풀이 · 해설 생성
현황: 프로토타입 운영 다음 단계: 대화 UX/컨텍스트 지속

SNargo 엔진

역할: 수능 특화 추론/계산 코어
현황: 수학 1000문항 중 약 2문항 오답 다음 단계: 과탐 → 사탐 → 국어 → 영어 확장

OpenEye (문제 인식 · 초지각)

역할: 문제 인식률 향상 (OCR+Vision)
현황: 인식 파이프라인 고도화 다음 단계: 도표/그림/수식 인식 강화

유사문제 출제기

역할: 수능·내신 문제 변형/생성
현황: 설계 완료 다음 단계: 베타 출시

프론트엔드 개발

역할: 사용자 인터페이스 · 웹/앱 개발
현황: SNarVIS 앱 UI/UX 개발 다음 단계: 반응형 웹 · 모바일 최적화

백엔드 개발

역할: 서버 · API · 데이터베이스 관리
현황: API 성능 최적화 진행 다음 단계: 확장성 개선 · 보안 강화

② 데이터 기반 프로젝트 SNarlink / Analytics

학습량 계측과 시각화, 자동 채점/분석으로 학생의 학습 행동과 성과를 정밀 측정합니다.

SNarlink (차단 + 계측)

역할: 인터넷 차단 · 학습량/접속 로그 수집
현황: 운영 중 다음 단계: 지점 확장 · 안정화

DataViz (Tableau)

역할: 학습 행동 시각화
현황: 간트차트(시간대별) · 크기맵(주요 사이트) 다음 단계: 성적 분석현황 DATA 시각화

OCR 채점

역할: 모의고사 자동 채점 → 오답률/취향 유형 분석
현황: 완성 단계 다음 단계: 누적 이력 기반 처방형 리포트

Bifrost

역할: AI 기반 방화벽 탐지/감시
현황: 개발 중 다음 단계: 실시간 위협 탐지 · 자동 차단

문제은행 시스템

역할: 평가원·교육청·교육업체·학교 내신 문제 DB화
현황: 설계 단계 다음 단계: 데이터 수집 · 분류 · 검색 시스템 구축

개인용 방화벽 판매 회사

역할: SNarlink의 방화벽 개인별, 학원별 판매 조직
현황: 제품 구상단계 다음 단계: 상용화
